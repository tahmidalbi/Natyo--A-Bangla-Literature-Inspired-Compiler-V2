Project Name: Natyo
Type: A narrative-driven Bangla-inspired mini programming language

Purpose:
Natyo is a small custom language for a compiler project. It is inspired by Bangla words and story/narrative structure. The goal is to make programming feel like directing a drama or story.

Core Idea:
A Natyo program starts like a story opening and ends like a story ending.

Main Structure:
- prostabona = beginning of the program
- uposhonghar = end of the program
- drissho = code block / scene
- shonkhya = integer data type
- purnoshonkhya = float data type
- shobdo = print output
- jobdo = input
- jodi = if
- onnotha = else
- chokro = loop

Special Features:
1. Narrative structure instead of traditional C-style syntax
2. Runtime history log in history_log.txt
3. Preloaded scientific constants:
   - PAI = 3.1416
   - E = 2.718
   - G = 9.8
4. Intrinsic math functions:
   - shakti(base, power)
   - borgomul(number)

What the language should support:
- variable declaration
- assignment
- arithmetic expressions
- printing
- conditionals
- loops
- simple math functions
